## Getting Started

1. create `.env.development` file in root folder with valuw of:

```javascript
SANITY_STUDIO_PROJECT_URL=YOUR_URL
SANITY_STUDIO_PREVIEW_SECRET=YOUR_SECRET
```

2.  run the development server:

```bash
npm run start
# or
yarn start
```
